# 动物

## 猎猫
- 六条腿
- 三只眼睛
- 肉食动物
- 外形类似于猫
- 黑色或深色花纹
- 生活在森林里
- 体积小
- 速度快，灵活
- 身高0.5米～1.5米，成年身高一般在1米左右
- 雄性体型明显大于雌性

## 盲牛
- 生活在荧光孢子环境中
- 无视力
- 通过超声波感知环境
- 草食动物
- 六足
- 双角
- 皮厚
- 行动缓慢
- 体积庞大
- 身高1米～5米，成年身高一般在3米左右
- 雌性体型明显大于雄性

## 地行虫

- 无视力
- 肉食动物
- 口中有6～200层环状牙齿（成年后平均在120层左右）
- 喜欢在地下潜行然后从下方窜出偷袭猎物
- 一般生存在沙漠、平原地区
- 直径1米～2米
- 无性别
- 体液为墨绿色，浓稠，带有凝结效果
- 体液有剧毒

## 灰兔

- 草食、昆虫为食
- 繁殖能力强
- 虽然名为兔，但是和兔子没啥关系
- 身高10CM～30CM
